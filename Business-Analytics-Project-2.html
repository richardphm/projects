Introduction

The data being analyzed in this project is fuel economy data collected by the U.S. EPA. The data shows vehicle manufacturer, year, model, class of vehicle, highway MPG, city MPG, etc. The goal is to use the provided dataset to analyze, transform, and visualize the data. From a first glance, there seems to be sufficient data to create transformations, visuals and to model.

Visualizations

Engine Cylinder Count vs City MPG

database %>%
  ggplot(aes(x = `City.MPG..FT1.`, y = `Engine.Cylinders`, stringr::str_wrap(V1, 15))) +
  geom_point() +
  geom_smooth(se = FALSE, method = "lm") +
  labs(title = "Relationship Between Engine Cylinder Count and City MPG", 
  x = "City MPG", y = "Engine Cylinder Count") +
  theme_light() +
  theme(plot.caption = element_text(hjust = 0))

In the scatter plot above, you can see that as Engine Cylinder Count decreases, City MPG increases. Fuel efficiency increases with fewer engine cylinders.

Highway MPG of Compact Cars Over Time

database %>%

  group_by(Year) %>%
  summarise(avg_mpg = mean(Highway.MPG..FT1.)) %>%

In the line graph above, it is clear that over time, Highway MPG of Compact Cars has improved. There were some years that the average MPG of the class dropped below previous years, but the average MPG begins to increase heavily around 2009.
